INNER JOIN Demonstration

View users with their orders using a JOIN query:

SELECT users.name, users.email, orders.item, orders.amount
FROM users JOIN orders ON users.id = orders.user_id View Joined Data

Database Schema

Users table:
  id INT PRIMARY KEY
  name TEXT
  email TEXT UNIQUE

Orders table:
  id INT PRIMARY KEY
  user_id INT (foreign key to users.id)
  item TEXT
  amount DECIMAL
